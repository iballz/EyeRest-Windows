This is a simple Eye resting app made for windows.

To use it,

1. make a directory for example : C:\Users\user_name\Documents\EyeRest
2. make a shortcut by holding Alt key and dragging the main.py file in the same folder.
3. go to the startup folder by pressing Win + R and typing shell:startup
4. move the shortcut file to the startup folder and you are good.

Icon of the app will be visible in your system tray.
currently there is only the option to exit the app, in future i will add other features also.
