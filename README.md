This is a simple Eye resting app made for windows.

To use it,

1. Download the zip file of this repository
2. make a directory for example : C:\Users\user_name\Documents\EyeRest.
3. Move all the files (icon.png, main.pyw and read.md) into the above made directory.
4. make a shortcut by holding Alt key and dragging the main.pyw file in the same folder OR by simply just right clicking the main.pyw file and then "more options" then create shortcut.
5. Then go to the startup folder by pressing Win + R and typing shell:startup
6. move the shortcut file to the startup folder and you are good. After restart the file will start on startup

Icon of the app will be visible in your system tray.
currently there is only the option to exit the app, in future i will add other features also.
